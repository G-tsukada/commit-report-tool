スーホルムの実績データを集計し、デザインされたHTMLレポートメールを送信してください。

## 処理フロー

### 1. メールデータを読み込む

```bash
cat /tmp/soholm-emails.json
```

### 2. メールが0件の場合

「対象日のメールが見つかりませんでした」というメールを送信して終了。

```bash
node scripts/soholm/send-report.js \
  --subject "【実績集計】スーホルム ${target_date} - 対象メールなし" \
  --body "対象日（${target_date}）の実績報告メールは見つかりませんでした。"
```

### 3. メール本文からデータを抽出する

各メールの本文を読み、以下の情報を店舗ごとに抽出してください。

**各店舗（最大3店舗）**
- 店舗名
- 予算（金額）
- 実績（金額）
- 達成率（実績 ÷ 予算 × 100、小数点第1位まで）
- トピックス・特記事項

**全体集計**
- 3店舗の合計予算
- 3店舗の合計実績
- 合計達成率
- 明日以降のアクション

データがない項目は「-」と記載し、推測・追加はしないこと。
複数メールで矛盾するデータがある場合は両方を記載し「※要確認」と付記する。

### 4. HTMLレポートファイルを生成する

抽出したデータを使って、Node.js で `/tmp/soholm-report.html` を生成してください。

**デザイン仕様**
- ページ背景: `#ffffff`（白）
- アクセントカラー: `#f59e0b`（アンバー・見出し・矢印・ヘッダーボーダー左線など）
- カード背景: `#e4dfd4`（グレージュ）※ヘッダー・各店舗・TOTAL・TOPICS・NEXT ACTIONS のブロックすべて
- メインテキスト: `#44403c` / `#292524`（見出しタイトル `#1c1917`）
- ラベル・補足: `#78716c` / `#57534e`
- 達成率バーのトラック（棒の裏）: `#c4bdb0`
- TOPICS の店舗名ラベル: `#4338ca`（紫）／左ボーダー `#6366f1`
- フッター区切り線: `#d6d0c4`
- 最大幅: 620px、中央揃え
- フォント: `-apple-system, BlinkMacSystemFont, 'Hiragino Sans', 'Yu Gothic', sans-serif`
- HTMLはインラインスタイルのみ使用すること（Gmail互換のため）

**達成率による色分け**
- 100%以上: `#10b981`（緑）
- 80〜99%: `#f59e0b`（オレンジ）
- 80%未満: `#ef4444`（赤）

**セクション構成（全4セクション）**

```
┌─────────────────────────────────────┐
│  SOHOLM DAILY REPORT                │  ← アンバーのボーダー付きヘッダー
│  スーホルム 実績レポート              │
│  対象日: XXXX-XX-XX ／ 生成: XX:XX  │
└─────────────────────────────────────┘

01 ／ STORE RESULTS
  ┌──────────────────────────────┐
  │ 店舗名                        │
  │ 予算  ¥XXX,XXX               │
  │ 実績  ¥XXX,XXX（色分け）      │
  │ ████████░░  達成率 XX%        │  ← 達成率バー
  └──────────────────────────────┘
  （店舗2、店舗3も同様）

02 ／ TOTAL
  ┌──────────────────────────────┐
  │ 合計予算 ¥XXX,XXX → 合計実績 ¥XXX,XXX   達成率 XX%  │
  └──────────────────────────────┘

03 ／ TOPICS
  ┌─ 店舗1 ────────────────────┐
  │ トピックス内容               │
  └─────────────────────────────┘
  （店舗2、店舗3も同様）

04 ／ NEXT ACTIONS
  →  アクション1
  →  アクション2
  →  ...

  🐱 コミネコ（スーホルム実績ツール）が自動生成
```

**HTML生成コマンド例（Node.jsのfs.writeFileSyncを使用）:**

```bash
node -e "
const fs = require('fs');
const html = \`<!DOCTYPE html><html lang='ja'>...(生成したHTML)...</html>\`;
fs.writeFileSync('/tmp/soholm-report.html', html, 'utf-8');
console.log('HTML生成完了');
"
```

HTMLテンプレートの注意点:
- テンプレートリテラル（バッククォート）を使う場合、内部のバッククォートは `\`` とエスケープすること
- 達成率バーの幅は `width: Math.min(達成率, 100)%` で指定すること
- 金額は `¥` 記号を付けてカンマ区切りで表示すること

### 5. HTMLレポートメールを送信する

```bash
node scripts/soholm/send-report.js \
  --subject "【実績集計】スーホルム {target_date}" \
  --html-file /tmp/soholm-report.html
```

件名の `{target_date}` は実際の日付（例: `2026-05-07`）に置き換えてください。

スーホルムの実績データを集計し、レポートメールを送信してください。

## 処理フロー

### 1. メールデータを読み込む

```bash
cat /tmp/soholm-emails.json
```

ファイルには以下の構造でデータが入っています：
```json
{
  "target_date": "2026-04-09",
  "email_count": 3,
  "emails": [
    {
      "subject": "【実績報告】【スーホルム】",
      "from": "...",
      "date": "Wed, 9 Apr 2026 20:15:00 +0900",
      "body": "実績データの本文..."
    }
  ]
}
```

### 2. メールが0件の場合

「対象日のメールが見つかりませんでした」というメールを送信して終了。

```bash
node scripts/soholm/send-report.js \
  --subject "【実績集計】スーホルム ${target_date} - 対象メールなし" \
  --body "対象日（${target_date}）の実績報告メールは見つかりませんでした。"
```

### 3. メール本文から実績データを抽出・分析する

各メールの本文を読み、以下の情報を抽出してください：

- **日付・時間帯**
- **売上・件数・金額** などの数値データ
- **商品・カテゴリ別** の内訳（あれば）
- **前日比・前週比・目標比** などの比較データ（あれば）
- **コメント・メモ** （スタッフや担当者のメッセージ）

複数メールがある場合は、同じ日のデータを統合してください。

### 4. レポート本文を生成する

`/tmp/soholm-report.txt` に以下の形式でレポートを書き出してください：

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🏪 スーホルム 実績集計レポート
対象日: {target_date}
生成日時: {now_jst}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【サマリー】
（数値の全体集計をここに記載）

【詳細データ】
（メールから抽出した詳細をここに記載）

【コメント・メモ】
（スタッフコメントや特記事項があれば）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🐱 コミネコ（スーホルム実績ツール）が自動生成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**記載ルール：**
- 数値データは見やすく整列させる（右揃え等）
- 複数メールのデータは重複なく統合する
- 欠損データは「-」と記載し、推測しない
- 専門用語はそのまま使用する（実績データは正確さ優先）

### 5. レポートメールを送信する

```bash
node scripts/soholm/send-report.js \
  --subject "【実績集計】スーホルム {target_date}" \
  --body-file /tmp/soholm-report.txt
```

件名の `{target_date}` は実際の日付（例: `2026-04-09`）に置き換えてください。

## 注意事項

- 数値は正確に転記する（四捨五入・省略禁止）
- メールに記載のない情報は追加しない
- 複数メールで矛盾するデータがある場合は両方を記載し「※要確認」と付記する
